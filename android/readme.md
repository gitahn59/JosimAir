# JosimAir Android

JosimAir의 애플리케이션은 측정기로부터 실내 공기 정보를 실시간으로 수신하여 사용자에게 제공한다.     
이 디렉토리에서는 안드로이드 애플리케이션의 소스코드와 관련 고지사항을 제공한다.
