# JosimAir

아두이노로 측정한 실시간 실내공기정보를 안드로이드로 송신하고
안드로이드는 외부 대기정보와 비교 후 사용자에게 환기 여부를 알려줍니다.

1. 외부 대기 정보 
2. 실내 공기 정보
3. 일간, 주간, 월간 통계
